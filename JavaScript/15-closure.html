1. Closure is the combination of function and lexical environment within which that function is declared.
  2. Closure is an action in which inner function can access to outer variable and all the global variables.

Uses

  • Module Design Pattern
  • Currying
  • Functions like once
  • memoize
  • maintaining state in async world
  • setTimeouts
  • iterators

DisAdvantages

  1. Memory usage : every time closure is created it allocates memory
